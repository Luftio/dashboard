# Luftio Dashboard

[![Cypress.io](https://img.shields.io/badge/tested%20with-Cypress-04C38E.svg)](https://www.cypress.io/)

## Getting started

```bash
$ npm install
$ npm run dev
```

## Building and deploying in production

```bash
$ npm run build
$ npm start
```

## Other

```bash
# storybook
```
