# Basic text

We have more types of basic text:

1. This is a default simple text.
2. This is a simple text used for dates in the Message card.
3. This is a simple text used for name in the Message card.
4. This is a simple text used for procents in the Message card.
5. This is a simple text used for empty state messages.
6. This is a simple text used in dashboard card for procents.
7. This is a simple text used dashboard card for naming columns.

export const Component = (args) => <BasicText>{args.text}</BasicText>;
export const ComponentDate = (args) => <BasicText date>{args.text}</BasicText>;
export const ComponentName = (args) => <BasicText name>{args.text}</BasicText>;
export const ComponentProcents = (args) => (
  <BasicText procents color={args.color}>
    {args.text}
  </BasicText>
);
export const ComponentEmptyState = (args) => <BasicText emptystate>{args.text}</BasicText>;
export const ComponentEvents = (args) => <BasicText events>{args.text}</BasicText>;
export const ComponentDashboardCardBottomProcents = (args) => (
  <BasicText bottomRowProcents color={args.color}>
    {args.text}
  </BasicText>
);
export const ComponentDashboardCardBottomName = (args) => <BasicText bottomRowProcentsName>{args.text}</BasicText>;
