# Buttons

In our low-level design system, we have currently 7 types of buttons.

1. Default that is used the most through our entire application.
2. Primary button that is used when submitting forms and to confirm steps in "danger zone".
3. Email button that is used with email clients.
4. Sign Me back button is used after user signs out (the styling is white so set your backgorund in Storybook to something more dareker to see it properly).
5. Save changes button that is used in forms that are in the dashboard section.
6. Navigation button on events page.
7. Active navigation button on events page.

You can see the examples right under this text.

export const Component = (args) => <Button>{args.text}</Button>;
export const ComponentButton = (args) => <Button primary>{args.text}</Button>;
export const ComponentEmail = (args) => <Button email>{args.text}</Button>;
export const ComponentSignMeBack = (args) => <Button signout>{args.text}</Button>;
export const ComponentSaveChanges = (args) => <Button savechanges>{args.text}</Button>;
export const ComponentNav = (args) => <Button nav>{args.text}</Button>;
export const ComponentNavActive = (args) => (
  <Button nav active>
    {args.text}
  </Button>
);
